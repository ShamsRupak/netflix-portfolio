# 🎬 Shams Rupak — Netflix-Inspired Portfolio

A Netflix-style interactive portfolio showcasing my work, skills, and experience as a **Software Engineer & AI/ML Engineer**.

**🔗 Live:** [Coming Soon]

## Features

- 🎬 **Netflix-style intro animation** with custom SR branding
- 👤 **4 profile types** — Recruiter, Engineer, Researcher, Explorer — each with curated content
- 📂 **6 project showcases** — StreamForge, TinyLang, Nexus, SentinelBoard, PulseAPI, NYC Weather
- 📊 **20+ skills** categorized across Languages, AI/ML, Web, and DevOps
- 📅 **Interactive timeline** for work experience & education
- 🎓 **Education page** with coursework details
- 📱 **Fully responsive** — mobile sidebar navigation

## Tech Stack

- ⚛️ **React** + **TypeScript**
- 🎨 **CSS3** with custom animations
- 🧭 **React Router** for navigation
- 📊 **react-vertical-timeline** for experience display
- 🎯 **Zero external CMS** — all content is self-contained

## Quick Start

```bash
git clone https://github.com/ShamsRupak/netflix-portfolio.git
cd netflix-portfolio
npm install
npm start
```

No `.env` file needed. No API keys. Just clone and run.
